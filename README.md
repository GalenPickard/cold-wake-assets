# cold-wake-assets

Static assets for the Datadog **Cold Wake** CTF narrative scenario (DASH 2026 booth).

Hosted via GitHub Pages — these pages are embedded as iframe widgets in the in-scenario Datadog dashboard.

## Files

- `mission-clock.html` — Ticking Y2K38 mission clock embedded in the Argos Mission Overview dashboard. Starts at `2038-01-19 03:14:07 UTC` and ticks one second per real second.

Source scenario lives in `DataDog/experimental` at `users/galen.pickard/ctf-booth-scenarios/`.
